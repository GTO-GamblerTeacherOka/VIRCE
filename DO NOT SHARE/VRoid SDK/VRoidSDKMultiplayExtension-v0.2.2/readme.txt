◆ VRoid SDK MultiplayExtensionのインストール
ダウンロードしたVRoidSDKMultiplayExtension.unitypackageファイルを

Assets > Import Package > Custom Package...

からインポートすることでインストールが完了します

VRoid SDK MultiplayExtensionはAssets/VRoidSDK/Extensions/Multiplay以下にインストールされます。
Assets/VRoidSDK/Extensions/MultiplayにREADME_ja.mdがあるので、こちらもご一読ください。

◆ アプリケーション登録とMultiplay ExtensionのSDK設定をおこなってください
◇ アプリケーション登録
https://hub.vroid.com/oauth/applications
事前に登録いただいたVRoid Hubアカウントへログインした状態で以下URLへアクセスすると、OAuthアプリケーション登録が行えます。
アプリケーション登録時にmultiplayスコープを設定することでマルチプレイ目的でのモデル利用の機能を利用することが可能になります

◇ MultiplayExtensionの設定
同梱しているREADME_ja.mdをご参照ください。
